学生目录使用说明


欢迎来到 学生目录 2.0！本页面将详细介绍如何使用学生目录的各项功能。


自学生目录创建以来，现已出具规模。但仍有许多访问者并不清楚学生目录的功能与使用方法。因此，我制作了此使用说明，以帮助访问者更好地使用学生目录。


简短来说，学生目录是一个部署在校园网内的，专供学生使用的平台。网站以为学生服务为宗旨，所有学生都可以享受学生目录的服务。


更多有关学生目录的信息，请参阅学生目录的“关于”页面。

如果你遇到任何问题，请将问题发送至留言簿或联系网站管理员

如何访问学生目录

学生目录部署在校园网内，使用想要访问学生目录，首先要接入校园网。

所有班级教室电脑在正常状态下都已经接入校园网，无需额外操作。

如果使用手机，则需要连接学校内的任意网络，连接网络即可，无需通过教师认证，有关教学楼中各楼层的WiFi密码，请参阅教程中的校园网密码登记

接入校园网后再浏览器中输入校园网网址即可访问学生目录。

学生目录2.0更新后ip变更为10.88.202.71，旧网址10.88.202.54将不再使用。


学生目录主界面

主界面仍在开发中，如果有什么建议或意见，请将您的意见发送至留言簿

主界面右侧按钮下方的圆圈代表当前校园网接入服务的质量，您可以通过颜色判断接入的状态，绿色代表良好，黄色代表中等，红色代表差。


校园网络接入服务

校园网络接入服务是学生目录的核心功能，它为学生提供校园网接入服务，并提供技术支持。

校园网络接入服务的质量直接影响到学生的上网体验，良好的接入服务可以让学生享受到校园网带来的便利，同时也能提高学生的上网体验。

关于如何使用校园网络接入服务，请参阅教程中的相关内容。


共享目录——文件共享服务

学生共享目录2.0更新后无需密码即可访问共享目录，登录账号后可查看更多文件

请注意：共享目录仅供学生使用，请勿泄露账号密码！若遗忘账号密码，请联系管理员

提示：好好利用目录中的搜索功能课可以更加方便高效地找到你想要的目标哦

以下为不同账号之间的权限划分：

账号类型 权限描述
游客登录 仅可访问共享目录中的公开文件，登录后可查看更多文件
学生账号 可上传文件至共享目录，可提供密码查看所有文件

期待您在学生目录中上传您的文件

留言簿

一个用于留言的网页，所有人都可以自由地发送与查看留言

留言内容将会被管理员审核，审核通过后才会显示在留言簿中

留言的发送是匿名的，不会显示发送者的真实姓名，发送时仅记录发送时间

留言簿支持切换主题功能，点击留言簿右上角的图标即可切换

请注意：请不要在留言簿中发布敏感信息
