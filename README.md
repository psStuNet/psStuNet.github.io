# psStuNet.github.io

这是一个学生目录的日志网站，用于展示学生相关资源、信息和学习资料。
关于学生目录请前往这里[Ps-Student-Catalog](https://ps-student-catalog-team.github.io/Ps-Student-Catalog/)

## 项目结构

- **article/**: 存放各类技术文章和教程
- **img/**: 图片资源文件夹
- **logs/**: 项目开发日志
- **media/**: 媒体文件（如播客）
- **index.html**: 网站首页
- **style.css**: 全局样式表

## 功能特点

- 响应式设计，适配各种设备
- 学生资源分类与多维度筛选系统
- 时间轴式学生活动日志展示
- 学生信息展示与管理
- 图片和媒体资源展示
- 交互式分类标签与过滤功能

## 使用方法

1. 克隆本仓库
```bash
git clone https://github.com/psStuNet/psStuNet.github.io.git
```
2. 打开index.html文件即可查看网站
